

# 📊 Road Accident Analysis Dashboard

**Abin R Nair**

## 📌 Project Overview
This project focuses on analyzing and visualizing road accident data to identify patterns, trends, and key factors affecting accident severity. An interactive Excel dashboard is built to support data-driven decision-making and improve road safety awareness.

---

## 🎯 Objectives
- Analyze road accident data to understand trends and patterns  
- Identify key factors influencing accident severity  
- Provide insights to improve road safety initiatives  
- Enable data-driven decision making  

---

## 📂 Dataset Information
- <a href = "https://github.com/Abin-2002/Excel-Road-Accident-dashboard/blob/main/Road_Accident_Data.csv">Dataset View </a>
- **Source:** Kaggle (Road Accident Dataset)  
- The dataset contains structured records of road accidents with multiple influencing factors.

### 🔑 Key Features:
- Accident Severity  
- Accident Date & Year  
- Light Conditions  
- Number of Casualties  
- Number of Vehicles  
- Road Type & Surface Conditions  
- Urban or Rural Area  
- Weather & Wind Conditions  
- Vehicle Type  

---

## ⚙️ Methodology
- Data cleaning (removed duplicates, handled missing values)  
- Standardized categorical fields (road type, weather, lighting)  
- Created calculated columns (Year, severity categories, total casualties)  
- Used Pivot Tables for analysis  
- Built an interactive Excel dashboard for visualization  

---

## 📊 Dashboard Features

### 🔹 KPIs
- Total Casualties  
- Fatal Severity  
- Serious Severity  
- Slight Injuries  
- Casualties by Vehicle Type  

### 🔹 Visualizations
- Donut Charts (Severity, Vehicle Type, Area)  
- Line Chart (Year-wise trend)  
- Bar Charts (Road Type analysis)  
- Tree Map (Road Surface conditions)  

### 🔹 Filters (Slicers)
- Date (Year, Month, Day)  
- Weather Conditions  
- Wind Conditions  

---

## 📈 Key Insights
- Cars contribute to the majority of accidents  
- Single carriageways are the most accident-prone  
- Wet road conditions increase accident severity  
- Daytime accidents are higher than nighttime  
- Urban areas report more accidents  

---

## 💡 Recommendations
- Implement stricter safety measures for car usage  
- Improve infrastructure on single carriageways  
- Enhance road drainage systems  
- Strengthen traffic control in urban areas  
- Focus on high-risk periods during the year
